To use the face attendance system:

Install requirements to run the app
1. pip install -r requirements.txt

2. Once requirements are installed open a command prompt and change directories to this folder

3. To run the app type in the command prompt this command:
streamlit run app.py

===================================================================
How to use the app:
Once the command is executed, the app should open in your default browser.

Select sign up from the dropdown menu in the sidebar to sign up with your username, password, and select an image of yourself to upload to the database_images folder to register yourself.

To take attendance, login with your credentials and upload an image of yourself. The app will run facial recognition and verification to identify if the user is valid, if not it will be detected as an unknown user.
